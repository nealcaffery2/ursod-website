## Troubleshooting

### If sitemap.xml or robots.txt return 404:
1. Check Vercel dashboard → Deployments → Latest deployment
2. Verify files are included in deployment
3. Check build logs for errors
4. Manually redeploy if needed

### If site shows "Not Secure":
1. Verify SSL certificate in Vercel dashboard
2. Check domain DNS settings
3. Wait 5-10 minutes for SSL to propagate
4. Clear browser cache

### If deployment fails:
1. Check build logs in Vercel dashboard
2. Verify `vercel.json` syntax is correct
3. Ensure all files are committed to Git
4. Check for any build errors
