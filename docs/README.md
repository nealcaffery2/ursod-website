# URSOD Company Website

A luxury, modern website for URSOD - an Oil & Gas Acquisition & Exploration company specializing in midstream ventures.

## Features

- **Hero Section**: Full-screen video background featuring skyscrapers
- **Luxury Design**: Black and white color scheme with elegant typography
- **Responsive Layout**: Fully responsive design for all devices
- **Smooth Animations**: Fade-in effects and smooth scrolling
- **Modern UI**: Clean, minimalist interface with premium aesthetics

## Sections

1. **Hero**: Video background with company introduction
2. **About**: Company overview and statistics
3. **Services**: Four main service offerings
4. **Approach**: Four-step process explanation
5. **Contact**: Contact form and information
6. **Footer**: Additional links and information

## Setup

1. Simply open `index.html` in a web browser
2. No build process or dependencies required

## Video Background

The hero section uses a video background from Pexels. If you'd like to use your own video:

1. Replace the video source in `index.html` (line 36) with your video URL
2. Supported formats: MP4, WebM
3. Ensure the video is optimized for web (compressed, appropriate resolution)

## Customization

- **Colors**: Edit CSS variables in `styles.css` (lines 13-19)
- **Content**: Update text content in `index.html`
- **Contact Information**: Modify contact details in the contact section
- **Company Information**: Update statistics and details in the about section

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Notes

- The contact form currently shows an alert on submission. Integrate with your backend/email service for production use.
- The video background includes a fallback to a static image if the video fails to load.
