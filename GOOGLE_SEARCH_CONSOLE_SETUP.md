# Google Search Console Setup Guide

## Sitemap Upload Instructions

### Step 1: Access Google Search Console

1. Go to [Google Search Console](https://search.google.com/search-console)
2. Sign in with your Google account
3. Add your property: `https://ursod.co`
4. Verify ownership (choose one method):
   - **HTML file upload** (download verification file from Google)
   - **HTML tag** (add meta tag to your site)
   - **DNS record** (add TXT record to your domain)
   - **Google Analytics** (if you have GA installed)

### Step 2: Submit Your Sitemap

1. Once verified, go to **Sitemaps** in the left sidebar
2. Enter: `sitemap.xml`
3. Click **Submit**
4. Google will start crawling your site

### Step 3: Verify Sitemap Status

- Check the sitemap status (should show "Success")
- Monitor how many URLs were discovered
- Check for any errors or warnings

## Monitoring

Check Google Search Console regularly for:
- **Coverage**: Which pages are indexed
- **Performance**: Search impressions and clicks
- **Issues**: Any crawl errors or warnings
- **Mobile Usability**: Mobile-friendly status
- **Core Web Vitals**: Page speed metrics

## Best Practices

1. **Update sitemap** when adding new pages
2. **Update lastmod dates** when content changes
3. **Monitor Search Console** weekly
4. **Fix any errors** Google reports
5. **Submit updated sitemap** after major changes
